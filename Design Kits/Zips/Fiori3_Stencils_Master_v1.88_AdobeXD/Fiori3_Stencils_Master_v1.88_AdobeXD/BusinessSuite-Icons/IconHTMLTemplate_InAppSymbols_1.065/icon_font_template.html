SAP Business Suite InApp Icon Font Testing Template (Beta testing) HTML Unicodes

The SAP Business Suite InApp icon font consists of scalable pictograms in the SAP illustrative style. The flat, simple shapes keep the UI simple and uncluttered. They are essentially vectors and so are easily resized without degradation. Usage of the SAP icon font should be given priority but can be used in conjunction with the standard environmental font within a given platform (e.g. Windows 8 and the Segoe UI Symbol font). SAP icon font utilizes Unicode in the E000-EFFF range to map each character. Since there are no official Unicode assignments in this range, using them creates no conflicts as long as the font can be read and integrated into the platform in question. The icon font has been created in .eot for compatibility with internet explorer, it is also available in .ttf format for other platforms and is available in other formats, if requested, to simplify embedding and integration.

This page shows a preview of the SAP Business Suite InApp icons. The font MUST be "hand hinted" (aligned) for each size so that it is optimized for all screen resolutions.
